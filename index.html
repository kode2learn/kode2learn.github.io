What is Kode2Learn?

Kode2Learn is a student organization at Miami University in Oxford Ohio. We organize and run events for kids in the Oxford area that promote teamwork and problem solving through the use of computer programming.

Mission Statement

To organize and run events for kids in the Oxford area that promote teamwork and problem solving through the use of computer programming. The goal is not for children to learn how to become computer scientists; it is about learning to learn.

Our Values

Creativity

Some people think coding is boring, but we think it's an avenue to help jump start creativity at an early age.

Problem Solving

At the core of computer science is problem solving, and learning code can help foster problem solving well beyond just a mouse and keyboard.

Team Work

We take coding to a new level by allowing kids to work together on code and learn team work skills, especially during our K2L Day workshops.

Excitement

We want to get kids excited about coding. We use technology every day, it's time to start getting excited about it.
